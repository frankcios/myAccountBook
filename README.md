# myAccountBook
![App Icon](https://raw.githubusercontent.com/frankcios/myAccountBook/master/myAccountBook/Image/Icon/Icon%403x.png)
--------------------------
* 使用Core Data, UserDefaults, AVFoundation等框架
* 使用第三方套件Charts繪製圖表
* 包含每月＆每日的消費記錄
* 每月金額加總
* 每月限制金額
* 點選時間標籤即可自訂時間
